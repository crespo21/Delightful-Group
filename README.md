# Delightful Group - Web Application

Professional cleaning and landscaping services website for DelightfulGroup.africa.

## 🚀 Quick Start

### Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Testing

```bash
# Run unit tests
npm test

# Run E2E tests
npm run test:e2e

# Run E2E tests with UI
npm run test:e2e:ui

# View test coverage
npm run test:coverage
```

### Code Quality

```bash
# Run linting
npm run lint

# Run autonomous investigation
npm run investigate

# Create GitHub issues from investigation
npm run create-issues
```
